How should the calculator use wider screens? (tablet / desktop)

Centered Card

Single centered column, max-width ~480–560px, elevated card with shadow. Clean focus, feels like a polished product widget. Same on all screen sizes.

Split: Filters + Price

Two-column on desktop: filters left, price result fixed right. Efficient use of space, result always visible. Collapses to single column on mobile.

Wide Centered Grid

Max-width ~800px centered, filters laid out in 2-column grid on desktop. More spacious, uses horizontal space. Price result spans full width below.
