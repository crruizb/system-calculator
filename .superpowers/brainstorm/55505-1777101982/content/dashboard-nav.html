Dashboard Navigation (Desktop)

How should the admin dashboard be structured on wider screens?

Top Navbar

Horizontal nav with logo + links + dark/light toggle. Full-width content area below. Compact, modern SaaS feel. Dark mode toggle lives in top-right.

Icon Sidebar

Narrow dark sidebar (icon-width) on desktop with nav icons + dark/light toggle at bottom. Collapses to bottom tab bar on mobile. More app-like, saves vertical space.
